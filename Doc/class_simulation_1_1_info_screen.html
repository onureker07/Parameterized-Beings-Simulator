Whether you're looking to expand your knowledge or simply want to have some fun, the Parameterized Being Simulator is a valuable tool for exploring the fascinating world of predator-prey relationships.

Detailed Description

InfoScreen class is a JFrame that shows the information of the simulation. It is used to show the information of the simulation in a table. It is updated every time the simulation is updated. It is also used to write the information in a file for later use.
